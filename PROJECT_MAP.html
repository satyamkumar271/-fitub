1) High-level Folder Map

FolderKaam (Hinglish)

2) Important Env / Config

Payments
  • RAZORPAY_KEY, RAZORPAY_SECRET, RAZORPAY_WEBHOOK_SECRET → Razorpay integration
Invoice / GST
  • INVOICE_COMPANY_NAME → invoice header me company name
  • INVOICE_GST_NUMBER → seller (aapka) GSTIN
  • INVOICE_GST_RATE → GST % used in checkout
  • INVOICE_ADDRESS_* → seller address invoice header me
Note
  • Gym GSTIN invoice me sirf tab aata hai jab payer user_type = gymowner.

3) Routes Summary (routes/web.php)

Ye major routes hain. Exact mapping dekhne ke liye routes/web.php open karein.

Public

  • / → FrontPageController@index (home)
  • /gyms, /trainers → directory listing pages
  • /profile/{user} → ProfileController@show (public profile + inquiry panel)
  • POST /inquiries → InquiryController@store (inquiry submit)

Auth Required

  • /dashboard → DashboardController@index
  • /dashboard/payments → DashboardController@payments + view dashboard/payments.blade.php
  • /billing/plans → PaymentController@plans + view payment/plans.blade.php
  • /invoice/{payment}/download → InvoiceController@userDownload
  • /support → SupportController (tickets)

Admin

  • /admin/dashboard → AdminController@dashboard
  • /admin/payments → AdminController@paymentsIndex + invoice download button
  • /admin/inquiries → AdminController@inquiriesIndex
  • /admin/invoice/{payment}/download → InvoiceController@adminDownload

7) Key Flows (simple steps)

Inquiry (Guest/Logged-in)

  • User public profile page open: /profile/{user} → view profile/show.blade.php
  • Send Inquiry button → slide-over form
  • Submit → POST /inquiries → InquiryController@store → Inquiry::create()

Billing / Plans / GST

  • /billing/plans → PaymentController@plans → view payment/plans.blade.php
  • Buy Now → POST /billing/order → Razorpay order create
  • GST rate config: INVOICE_GST_RATE. Payment record me breakup store hota hai.

Invoice Download

  • User side: /invoice/{payment}/download (only own paid payment) → PDF download
  • Admin side: /admin/invoice/{payment}/download (any paid payment)
  • Seller details: INVOICE_GST_NUMBER + INVOICE_ADDRESS_*
  • Gym buyer details: only if payer is gymowner → gym GST + gym address shown
